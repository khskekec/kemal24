# Kemal 24

Web-based diabetes management software developed for Kemal Kekec.

## Dashboard

* Day average
* Time in range
* Highest and lowest value
* Actual trend
* Day diagram

* Latest Bolus event (time until now)
* Latest correction event (time until now)
* Total Bolus
* Total Carbs
* Diff to Basal in percent (60 / 40 ?)
